RinggitPay payment gateway plugin for WooCommerce

== Description ==
Now you can integrate your WooCommerce shopping cart with RinggitPay Payment Gateway Malaysia in few simple steps

Search RinggitPay in plugin directory. Install and activate RinggitPay to accept payments online via **FPX – Internet Banking, Visa/Master Credit & Debit Cards**  

If you do not have a RinggitPay account, [Sign Up](https://ringgitpay.biz/sign-up/) to start accepting payments online

[Contact](https://ringgitpay.biz/sign-up/) our careline team for more information

Follow us on our [Facebook Page](https://www.facebook.com/RinggitPay) to stay updated on the latest releases and news.


== Installation ==

Before you begin, ensure to install and activate WooCommerce plugin to your WordPress website.

**Step 1:**
To add and activate RinggitPay plugin
- Go to: **Plugins > Add New**
- Search **RinggitPay** in plugin directory
- Click **Install Now > Activate Plugin**

**Step 2:**
To set up RinggitPay as your payment channel
- Go to: **WooCommerce > Settings > Payments > Enable RinggitPay**
- Click **Manage** to configure
- Configure gateway and provide your **unique App Id, Request and Response Key**
- Select **Production** from the ‘UAT/Production’ drop down
- **Save** changes to go live

[Click here](https://ringgitpay.biz/sign-up/) to get your unique code. Don’t have an account? [Sign Up](https://ringgitpay.biz/sign-up/) to get one  


**Congratulations!** Your customers can now place orders and shop from your website safely and securely with RinggitPay

For more details and queries [Contact Us](https://ringgitpay.biz/sign-up/)

**Test Transactions:**
You can switch to **UAT mode** to test transactions using our payment simulators and check whether RinggitPay is successfully integrated with your WooCommerce cart. 

Refer RinggitPay Integration Document to perform test transactions.

**Note:** Make sure to disable **UAT mode**, once your testing is completed. Switch to **Production** mode to go live.
